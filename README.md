#Análise de Margem e Rentabilidade por Centro de Custo

📌 Sobre o Projeto
Este projeto visa analisar a rentabilidade operacional de diferentes unidades de negócio ou centros de custo, identificando onde a empresa está gerando valor e onde a margem está sendo corroída por ineficiências operacionais.

🛠 Tecnologias UtilizadasPython (Pandas): Consolidação de bases de dados de receita e custos, cálculo de margem EBITDA e margem líquida por unidade.  SQL: Extração e tratamento de dados de faturamento e despesas a partir de estruturas relacionais.  Data Storytelling: Visualização de KPIs para suporte à decisão estratégica.

🚀 O que este script faz?Importa dados de faturamento bruto e despesas (operacionais e administrativas).  Calcula a Margem EBITDA ajustada para cada centro de custo.  Identifica unidades com performance abaixo da média corporativa (análise de underperformers).  Gera um relatório executivo que destaca a correlação entre receita e nível de despesa.

📈 Impacto EsperadoIdentificação precisa de centros de custo que necessitam de revisão orçamentária imediata.  Suporte fundamentado para a alocação de recursos em unidades com maior potencial de crescimento.  Agilidade no processo de Business Review mensal com a diretoria financeira.
